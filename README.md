# MKF Aghal Food App
### by HillTech Solutions

## Files in this folder
- `index.html` — Landing page (opens on GitHub Pages)
- `mkf-aghal.html` — Customer ordering app
- `mkf-owner-dashboard.html` — Owner dashboard
- `manifest.json` — PWA manifest (customer app)
- `manifest-dashboard.json` — PWA manifest (dashboard)
- `sw.js` — Service worker (offline support)

## GitHub Pages Setup Steps

1. Go to github.com → Sign in or create free account
2. Click **"New repository"**
3. Name it: `mkf-aghal` (or any name)
4. Set to **Public**
5. Click **"Create repository"**
6. Upload ALL files from this folder
7. Go to **Settings → Pages**
8. Under "Source" select **"main"** branch → **"/ (root)"**
9. Click **Save**
10. Your app will be live at:
    `https://HillTechSolutions.github.io/mkf-aghal/`

## Installing as App (PWA)
Once live on GitHub Pages:
- **Android:** Open in Chrome → Menu → "Add to Home Screen"
- **iPhone:** Open in Safari → Share → "Add to Home Screen"  
- **Desktop:** Open in Chrome → Address bar → Install icon
